/api/users/{id}			gets profile information
/api/users/{id}/songs		gets songs uploaded by user
/api/users/{id}/playlists	gets playlists created by user

/api/me		returns current user id
/api/liked	returns current user liked songs

/api/song/like?id={songID}	adds the song to liked
/api/song/dislike?id={songID}	removes the song from liked

/profile/edit		updates the profile information
/playlists/create	creates a new playlist
